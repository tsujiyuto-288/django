# テンプレートについて

## 変数の表示（{{ }} について）

ビュー（`views.py`）から渡された変数やオブジェクトのデータを、HTML上に展開して表示する。

### 💡 概要

- `{{ 変数名 }}` のように、変数を二重の中括弧で囲むことで、その中身を文字列としてWebページに出力できる。
- オブジェクトの属性、辞書のキー名、リストのインデックスなどにアクセスする場合は、Pythonのようなブラケット `[]` ではなく、すべて **ドット（`.`）** で繋ぐ。

### ✒️ 基本的な書き方

```html
<!-- 1. オブジェクトの属性（プロパティ）にアクセスする場合 -->
<!-- Pythonでの book.title と同じようにドットで繋ぐ -->
{{ book.title }}

<!-- 2. リスト（配列）やクエリセットのN番目にアクセスする場合 -->
<!-- Pythonでの book_dict[2].title ではなく、[ ]は使わずドット(.)の後にインデックス（数字）を書く -->
{{ book_dict.2.title }}

<!-- （参考）辞書（dict）のキーにアクセスする場合も同様 -->
<!-- Pythonでの data["name"] ではなく、ドットで繋ぐ -->
{{ data.name }}
```
